## Install the requirements / Instala los requisitos

```bash
python -m pip install -r ./requirements.txt
```

## Audio setup / Configuración de audio

- On the microphone input, select your microphone or the audio source you want to receive and decode.
- You can use Voicemeeter or virtual cables to route both microphone and system audio.

## Notas en español

- En la entrada de micrófono, selecciona tu micrófono o la fuente de audio que quieras recibir y decodificar.
- Puedes usar Voicemeeter o cables virtuales para enrutar tanto el micrófono como el audio del sistema.
